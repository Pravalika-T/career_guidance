# **App Name**: CareerCraft 3D

## Core Features:

- Interactive 3D Welcome: Engaging 3D animated welcome sequence with a friendly guide to introduce the app.
- Gamified Interest Discovery: Users select interests via floating 3D interactive cards that lead to personalized career recommendations.
- Intelligent Career Mapping: Maps user interests to 8 career domains containing 105 career paths.
- Smart Career Recommendations: The LLM acts as a tool and leverages user-selected interests to provide dynamically generated top career recommendations.
- Career Path Exploration: Step-by-step exploration of each career path, covering key aspects like roles, eligibility, skills, and salary.
- Progress Tracking & Gamification: Gamification elements like XP points, badges, and progress bars to motivate and track career discovery.
- Interactive 3D Roadmap: A 3-stage interactive roadmap for each of the 105 career paths, visually guiding the student from school to college to job with 3D animation, motion cues, and progressive disclosure.

## Style Guidelines:

- Primary color: A calming yet energetic blue (#4CAF50) to inspire confidence and exploration.
- Background color: A desaturated light blue (#E3F2FD) to provide a calming backdrop.
- Accent color: A vibrant purple (#9C27B0) to highlight interactive elements and CTAs.
- Body and headline font: 'Inter', a grotesque-style sans-serif, offering a modern and neutral feel.
- Custom 3D icons representing each career domain and path, enhancing visual engagement.
- Clean, minimal layout with a focus on smooth transitions and progressive information reveal.
- Micro-animations on every user action and smooth transitions to guide attention without distraction.
- Headings: clean, geometric font; medium weight.
- Descriptions: light weight, neutral font.
- Labels: compact & subtle font, small caps or reduced opacity.
